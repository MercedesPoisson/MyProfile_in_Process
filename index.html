ABOUT

I am a dedicated and aspiring full stack developer with a passion for building innovative web applications. Currently, I am on an exciting journey to become a proficient full stack developer, honing my skills in JavaScript, React, Redux, Express, Node.js, PostgreSQL, Sequelize, HTML, CSS, Bootstrap, Tailwind, and TypeScript.

With a strong foundation in JavaScript, I enjoy crafting interactive and dynamic user interfaces using React and Redux. Additionally, I have experience in developing server-side applications using Express and Node.js, along with working with databases like PostgreSQL and utilizing ORM tools like Sequelize.

My knowledge extends to front-end technologies, including HTML, CSS, and popular CSS frameworks like Bootstrap and Tailwind, allowing me to create visually appealing and responsive web designs. Furthermore, I am actively expanding my expertise to include TypeScript, as it provides strong typing and enhances code reliability.

I am driven by a constant desire to learn and keep up with the ever-evolving world of web development. I am excited about the challenges and opportunities that come with being a full stack developer, and I am eager to contribute my skills and enthusiasm to create engaging and impactful web experiences.
